10. Cross-Group Statistical Comparisons

Effect sizes are NOT on a common scale across groups. All comparisons below use formal statistical tests on MCMC posterior draws — not descriptive summaries of point estimates. Scale-invariant metrics only.

A. I² Heterogeneity — Study-Level Variance Decomposition

I² = τ²/(τ²+SE²) per variable, computed from MCMC posterior draws of τk. Each bar = median I² with 95% CrI error bars. Higher = more unexplained between-study variance relative to within-study noise. Formally comparable across groups because I² is dimensionless.

B. P(τg1 > τg2) — Pairwise Heterogeneity Comparison

For each MCMC draw, compute the median τ across all variables in each group, then count how often group 1's median τ exceeds group 2's. This is a fully Bayesian pairwise test — not a comparison of point estimates. Values near 0.50 = no difference; >0.80 or <0.20 = credible difference.

C. Posterior Predictive Checks — Model Adequacy

Two discrepancy measures per group: max |standardized residual| and χ² goodness-of-fit. For each MCMC draw, compute the discrepancy on observed data and on replicated data; the p-value = P(Trep ≥ Tobs). Values near 0.50 = good fit; <0.05 or >0.95 = misspecification. These are formally valid Bayesian p-values, not ad hoc.

D. Grade → Rank Spearman ρ — Does Evidence Quality Predict BNMA Rank?

For each MCMC draw, rank variables by μk and compute Spearman ρ with the ordinal grade score (A=4, B=3, C-PR=2, C-WP=1). The distribution of ρ across draws gives a fully Bayesian credible interval. ρ ≈ 0 means the BNMA ranking has decoupled from prior grade assignment — the data are speaking, not the labels.

E. Evidence → Precision Meta-Regression

Spearman correlation between n_studies and posterior SD across all 34 variables, with bootstrap 95% CI. Tests whether more replication evidence actually narrows posteriors in this BNMA. A significant negative ρ would validate the replication-augmentation strategy.

F. Permutation Test — Flag Rate Differences

Kruskal-Wallis test across all 4 groups (H=7.76, p=0.051), plus 10,000-permutation pairwise tests. The only significant differences (p<0.05) are S vs Overlay and R vs Overlay — both driven by Overlay being 100% Tier 3 candidates. S vs R and T vs R are NOT significantly different despite looking different in the descriptive charts.

G. Group Leaders — Posterior Overlap Coefficients

KDE-based overlap coefficient ∫min(f₁,f₂) between the #1 variable in each group. OVL near 0 = distributions well-separated; near 1 = indistinguishable. ⚠ Different scales — OVL measures distributional overlap, but the magnitudes are not comparable across groups. Useful for: assessing how precisely each group's leader is identified vs another group's leader's precision.

H. Tier Composition & Grade Distribution

Context for interpreting all the above: which groups have enough Grade A evidence to produce reliable posteriors? S has 9 Grade A variables; Overlay has zero. This explains why Overlay's PPC p-values are borderline and its I² is prior-dominated.
