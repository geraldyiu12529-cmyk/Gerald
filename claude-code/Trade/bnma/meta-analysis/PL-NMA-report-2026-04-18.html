Methodology

Model. Bradley-Terry pairwise reduction of the Plackett-Luce partial-ranking likelihood. For each paper p, the within-paper ordinal scalar is p_beats_peers (BNMA papers) or a grade-Sharpe composite (systematic reviews). Each within-paper ranking is decomposed into strict pairwise comparisons. Likelihood: P(i beats j) = σ(θ_i − θ_j). Prior: θ_i ~ Normal(0, 1) for every variable. No fixed reference variable; identifiability from the prior alone.

Inference. L-BFGS-B MAP optimisation, then Laplace approximation θ ~ Normal(θ*, H⁻¹). 10,000 posterior draws for rank/dominance summaries. Hessian min eigenvalue = 1.00 (log-concave posterior).

Scale caveat. The BNMA papers warn that cross-group posterior medians are NOT on a common scale. Using p_beats_peers (a probability) as the ordinal input is more scale-invariant than using the raw posterior median, but cross-group comparisons within a single paper still assume "beats peers in group X" is exchangeable with "beats peers in group Y". This assumption is weakest for Overlay (tiny peer set) and for small-R groups. Sensitivity S2 drops this assumption and re-fits with within-group BNMA comparisons only.

Limitations

  1. Cross-group commensurability assumption (see caveat above). S2 exposes two cases (V014, Faber TAA) where the primary fit overstates strength.
  2. n=1 candidates have wide credible intervals; treat as provisional.
  3. Laplace approximation, not full MCMC. For this log-concave posterior the approximation is good, but very tail probabilities (P < 0.02) would benefit from HMC.
  4. Paper-level heterogeneity not modelled — noise on θ, not a random effect.
  5. P2 and P3 are analytical approximations; their approx0 entries are excluded. S1 sensitivity drops these papers entirely and confirms the top-10 is robust.
