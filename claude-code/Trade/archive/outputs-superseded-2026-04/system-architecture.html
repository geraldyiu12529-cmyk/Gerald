🛡 Risk Rules (binding)

● Sizing: inverse-ATR vol-target, ¼ Kelly, 25% single-position cap
● Per-trade risk: 0.5–2%; portfolio heat ≤ 8%
● Stops: 2-3× ATR trailing (not fixed %)
● Circuit breaker: −15% → 50% exposure; −20% → defensive
● Correlation gate: co-moving theses share sector cap
● Time invalidation: no move by deadline → close

Pre-Entry Checklist

✓ |Sum| ≥ 3
✓ Invalidation written + dated
✓ Correlation gate clean
✓ Risk ≤ 2%, heat ≤ 8%
✓ ATR stop set
✓ No Grade C dependence
